Web 上有很多过时的信息，它们会导致新的 PHP 使用者误入歧途，传播不好的实践以及不安全的代码。PHP: The Right Way 是 PHP 流行的编程规范、
Web 上遍布的权威教程的链接以及贡献者当下认为的最佳实践的一份易于阅读的，快速的参考。

没有使用 PHP 的标准方法。本网站旨在向新的 PHP 开发人员介绍一些可能迟迟没被发现的主题，而且旨在为经验丰富的专业人士提供一些关于那些他们已经
进行多年却从未重新考虑的话题的新想法。这个网站将不会告诉你使用哪种工具，而是提供多种选项的建议，尽可能的解释方法以及用例上的差异。

这是一份活着的文档，并将持续的更新，去包括更多有用的信息和示例，一旦它们变得可用。
